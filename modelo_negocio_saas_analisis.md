
**F√≥rmula simplificada (clientes homog√©neos):**
```
MRR = N√∫mero de clientes √ó Ingreso promedio por cliente

Ejemplo:
10 clientes √ó $150/mes promedio = $1,500 MRR
```

### 13.1.3 C√°lculo de ARR (Annual Recurring Revenue)

**F√≥rmula:**
```
ARR = MRR √ó 12

Ejemplo:
MRR: $1,500
ARR: $1,500 √ó 12 = $18,000
```

**Ajuste por crecimiento:**
```
ARR proyectado = (MRR actual √ó (1 + Tasa de crecimiento mensual)^12)

Ejemplo:
MRR actual: $1,500
Crecimiento: 5% mensual
ARR proyectado = $1,500 √ó (1.05)^12 = $2,689/mes √ó 12 = $32,268
```

### 13.1.4 C√°lculo de Punto de Equilibrio

**F√≥rmula:**
```
Punto de Equilibrio (paquetes) = Costos fijos mensuales / Margen de contribuci√≥n por paquete

Donde:
Margen de contribuci√≥n = Precio - Costo variable

Ejemplo:
Costos fijos: $2,420/mes
Precio: $0.05
Costo variable: $0.012
Margen: $0.05 - $0.012 = $0.038
PE = $2,420 / $0.038 = 63,684 paquetes/mes
```

**En n√∫mero de clientes:**
```
Clientes necesarios = PE en paquetes / Paquetes promedio por cliente

Ejemplo:
PE: 63,684 paquetes
Promedio por cliente: 3,000 paquetes
Clientes: 63,684 / 3,000 = 21.2 ‚âà 21 clientes
```

### 13.1.5 C√°lculo de LTV (Lifetime Value)

**F√≥rmula simple:**
```
LTV = Ingreso mensual promedio √ó Tiempo promedio como cliente (meses)

Ejemplo:
Ingreso promedio: $150/mes
Tiempo promedio: 36 meses (3 a√±os)
LTV = $150 √ó 36 = $5,400
```

**F√≥rmula con churn:**
```
LTV = (Ingreso mensual √ó Margen bruto) / Churn rate mensual

Donde:
Churn rate mensual = Churn rate anual / 12

Ejemplo:
Ingreso mensual: $150
Margen bruto: 76% = $114
Churn anual: 15%
Churn mensual: 15% / 12 = 1.25%
LTV = $114 / 0.0125 = $9,120
```

### 13.1.6 C√°lculo de CAC (Customer Acquisition Cost)

**F√≥rmula:**
```
CAC = Gastos de marketing y ventas / N√∫mero de clientes adquiridos

Ejemplo:
Gastos de marketing (mes): $3,000
Clientes adquiridos: 5
CAC = $3,000 / 5 = $600 por cliente
```

**CAC Payback Period:**
```
Meses para recuperar CAC = CAC / (Ingreso mensual √ó Margen bruto)

Ejemplo:
CAC: $1,500
Ingreso mensual: $150
Margen bruto: 76%
Payback = $1,500 / ($150 √ó 0.76) = 13.2 meses
```

### 13.1.7 Ratio LTV/CAC

**F√≥rmula:**
```
Ratio LTV/CAC = LTV / CAC

Benchmark:
< 1: Insostenible (pierdes dinero por cada cliente)
1-3: Riesgoso (necesitas mejorar)
3-5: Saludable ‚úÖ
> 5: Excelente (pero quiz√°s gastas poco en adquisici√≥n)

Ejemplo:
LTV: $5,400
CAC: $1,500
Ratio = $5,400 / $1,500 = 3.6 ‚úÖ Saludable
```
